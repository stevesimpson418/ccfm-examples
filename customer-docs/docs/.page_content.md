# Nexus Platform Documentation

Welcome to the Nexus Platform developer documentation. Nexus is a B2B commerce platform API — use it to build order management, inventory, and fulfilment workflows into your applications.

> [!info]
> Looking for the status page or support? Visit [status.nexus.io](https://status.nexus.io) or email support@nexus.io. This documentation covers the API and integration guide only.

## API base URL

```
https://api.nexus.io/v2
```

All requests must use HTTPS. HTTP requests are rejected with `301 Moved Permanently`.
