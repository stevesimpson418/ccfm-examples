# Acme Corp Wiki

Welcome to the Acme Corp internal wiki — our single source of truth for company culture and team information.

> [!info]
> This wiki is managed as code in Git. To contribute, open a pull request against the `docs-wiki/` directory. Changes are applied to Confluence automatically on merge to `main`.

## How to navigate

| Section | What you'll find |
| --- | --- |
| [People & Culture](page:"People & Culture") | Team structure, handbook, policies |

## Contributing

1. Clone the repo and create a branch
2. Edit or add Markdown files under `docs-wiki/`
3. Run `ccfm --config ccfm-wiki.yaml plan --directory docs-wiki` locally to preview
4. Open a PR — CI will show you what pages will change
5. Merge to `main` — CI applies automatically
