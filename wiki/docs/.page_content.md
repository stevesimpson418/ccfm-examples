# Acme Corp Wiki

Welcome to the Acme Corp internal wiki — our single source of truth for engineering standards, product direction, and company culture.

> [!info]
> This wiki is managed as code in Git. To contribute, open a pull request against the `docs/` directory. Changes are deployed to Confluence automatically on merge to `main`.

## How to navigate

| Section | What you'll find |
| --- | --- |
| [Engineering](page:"Engineering") | Coding standards, onboarding, architecture decisions |
| [Product](page:"Product") | Roadmap, release notes, feature specs |
| [People & Culture](page:"People & Culture") | Team structure, handbook, policies |

## Contributing

1. Clone the repo and create a branch
2. Edit or add Markdown files under `docs/`
3. Run `ccfm --directory docs --plan` locally to preview the Confluence impact
4. Open a PR — CI will show you what pages will change
5. Merge to `main` — CI deploys automatically
